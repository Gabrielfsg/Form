## 📋 Sobre o projeto

Página com **formulário responsivo** construído apenas com HTML5 e CSS3, sem frameworks. Foco em layout adaptativo (mobile e desktop) e estilização de inputs.

## 🛠️ Tecnologias

- **HTML5**
- **CSS3** — Flexbox, Media Queries

## 📁 Estrutura

```
.
├── index.html
├── home.css
├── styles.css
├── responsive.css     # Regras específicas para telas menores
└── assets/            # Imagens e ícones
```

## 🚀 Como executar

Abra o `index.html` direto no navegador, ou rode um servidor local:

```bash
npx serve .
```

## 🎯 Aprendizados

- Layout responsivo com Media Queries
- Estilização de campos de formulário
- Boas práticas de HTML semântico
